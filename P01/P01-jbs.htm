Introduction to MPIDE 		 


Introduction

In this project, you will be introduced to MPIDE (the Multi-Platform Integrated Development Environment) which is software that can be used to program chipKIT boards. Programs written for MPIDE are called “sketches”. (We will often use the term code or computer code to refer to all or part of a sketch. Note that this “code” has nothing to do with a secret code even though some of it my initially appear somewhat mysterious!) In order to demonstrate how to use MPIDE with your chipKIT board, you will run one of the example sketches that is provided within MPIDE.

A brief discussion of the components used in this project is provided below. Links to slightly more detailed descriptions of the various components are provided in the subsections below.

  3. MPIDE can be use with a great variety of board but it needs to know the type of board it is supposed to “talk” to. To do this, as indicated below, select the menu Tools>Board and the type of your particular chipKIT board.

    Next, go to Tools>Serial Port and select the proper serial port.

    This is usually usually COM3 or higher. An easy way to learn which serial port is the correct one is to disconnect your chipKIT board. The correct option will have disappeared from the dropdown menu. Reconnect your board and select that serial port.



Step 2: Running the Sketch

We are now almost at the point where we can run the sketch. Normally, before uploading the sketch, you would “verify“ the sketch to identify any bugs or errors in the code. This can be accomplished by in on of three way. As indicated in the figure below, you can click on the button corresponding to Verify. Alternatively, you can click on the menu entry Sketch>Verify/Compile or you can type Ctrl+R (i.e., hold down the control key and then type the key R).

Whether or not a sketch as been verified, we are free to try to “upload” it to a board. If there is a bug in the code that would have been caught in the verification stage, that will still be caught when we try to upload it. To upload the sketch, you can click on the Upload button, as indicated above, click on the menu entry File>Upload to I/O Board (I/O stands for input/output), or type Ctrl+U. Once the upload process starts, two other light emitting diodes on the board, technically known as LED1 and LED2, should start to blink.
