Variable Types:

In algebra variables can only represent numbers. Programing variables can represent a lot more than numbers so it is important to let the computer know what type of variable it is working with. Figure 1 shows how to define a variable and its type as well as  a list of some basic types that can be used in MPIDE.


Fig. 1. Variable instantiation.

Type: int
The int type is an abbreviation for integer. It can be used to represent a positive or negative non-decimal number. It can be no greater than 2,147,483,647 and no less than -2,147,483,648.
Ex.
      Int myvar = -15;

Type: float
The float type is an abbreviation for floating point number. It is used to represent positive or negative decimal numbers. The number can have a maximum of 7 decimal digits.
Ex.
      float myvar = 15.0;
      float myvar = -15.1526;

Type: double
The double type is an abbreviation for double precision floating point number. It is used to represent positive or negative decimal numbers. The number can have a maximum of 15 decimal digits.
Ex.
 double myvar = 312.165;

Type: bool
The bool type is an abbreviation for Boolean. Boolean types can represent True or False.
Ex.
       bool myvar = true;
       bool myvar = false;


Type: char
The char type is an abbreviation for character. It can represent any single character highlighted red in the Fig 2. NOTE: Character values must be surrounded by single quotes ‘ ‘.
Ex.
      char myvar = ‘C’;
      char myvar = ‘1’;
      char myvar = ‘$’;
